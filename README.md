## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Vidhi-Mathur/Candidate_Requisition.git
    ```
2. Navigate to project repository:
    ```bash
   cd TIF_ React Developer Hiring Task-2 Repository
    ```
3. Install dependencies
    ```bash
   npm install
    ```
4.  Start server 
    ```bash
    npm run dev
    ```
